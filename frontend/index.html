How It Works

Four-stage serverless pipeline on AWS Free Tier — from PDF to structured insights in under 30 seconds.

01
Step 01

Extract Text

PyMuPDF pulls raw text from the PDF. pdfplumber fallback handles complex layouts and tables.

02
Step 02

Analyze with AI

Nemotron 120B receives text + a strict JSON schema prompt, returning structured data via OpenRouter API.

03
Step 03

Parse & Structure

Validated JSON output: name, email, skills, work history, education, and a recruiter summary.

04
Step 04

Store & Retrieve

Results persist in DynamoDB, served via API Gateway — accessible live in the dashboard below.
