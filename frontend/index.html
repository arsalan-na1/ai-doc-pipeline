How It Works

Four-stage serverless pipeline on AWS Free Tier — from PDF to structured insights in under 30 seconds.

STEP 01

Extract Text

PyMuPDF pulls raw text from the uploaded PDF, with pdfplumber as a fallback for complex layouts.

STEP 02

Analyze with AI

GPT-4o-mini receives the text and a strict JSON schema prompt, returning structured data via the OpenAI API.

STEP 03

Parse & Structure

The model returns validated JSON: name, email, skills, work history, education, and a recruiter summary.

STEP 04

Store & Retrieve

Results persist in DynamoDB and are served via API Gateway, accessible in the live dashboard below.
